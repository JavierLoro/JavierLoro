### 🧩 Sobre mí

Diseño y opero un **homelab propio sobre Proxmox VE** (+10 contenedores LXC/VMs en producción: reverse proxy, DNS interno, VPN _mesh_ con Tailscale, CI/CD) sobre el que despliego mis propias aplicaciones web. **Integro IA de forma sistemática en mi flujo de trabajo** — agentes y asistentes conectados a mis servicios vía **MCP (Model Context Protocol)**, automatización de despliegues y documentación, e informes inteligentes dentro de mis productos — usándola como multiplicador de productividad para construir y mantener todo el ciclo de forma autónoma.
