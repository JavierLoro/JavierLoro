### 🧩 About me

I design and operate a personal homelab on **Proxmox VE** (10+ LXC containers/VMs in production: reverse proxy, internal DNS, _mesh_ VPN with Tailscale, CI/CD) where I deploy my own web applications. **I integrate AI systematically into my workflow** — agents and assistants connected to my services via **MCP (Model Context Protocol)**, deployment automation and documentation — using it as a productivity multiplier to build and maintain the full cycle autonomously.

### 🏠 Personal infrastructure

I run a home server on **Proxmox VE** with 10+ LXC containers in production, each with a specific role: TLS reverse proxy, internal DNS, _mesh_ VPN (Tailscale), storage, and several self-built services.

My usual workflow: develop in a dedicated **devbox** → publish Docker image to GHCR → automatic deployment via **Watchtower** in the production container. Everything exposed under custom domains (`*.jlc-dev.me`) through the reverse proxy.

All architecture documentation lives in a **self-hosted wiki** ([wiki.jlc-dev.me](https://wiki.jlc-dev.me)) integrated with AI agents via MCP.
